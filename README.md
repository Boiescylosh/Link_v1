TESTED ON TERMINAL

### Requirements
- Node.js
- Git
  
### Installation ( Termux )
1. pkg update && pkg upgrade
2. pkg install nodejs git -y
3. git clone https://github.com/Boiescylosh/Link_v1
4. cd Link_v1
5. npm install
6. node Link_v1.js

### Installation ( Linux )
1. sudo apt install nodejs npm
2. git clone https://github.com/Boiescylosh/Link_v1
3. npm install 
4. node Link_v1.js

### Installation ( CMD/Power Shell )
1. Install dulu node.js di browser
2. node Link_v1.js
